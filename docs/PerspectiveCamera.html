Class: PerspectiveCamera

PerspectiveCamera

This is a structure that holds settings of perspective camera. If you want to switch viewer to use perspective camera set camera to 'perspective'. You can modify this but it is not necessary because sensible values are defined when geometry model is loaded with load() method. If you want to change these values you have to do it after geometry is loaded.

Constructor

new PerspectiveCamera()

This is only a structure. Don't call the constructor.
