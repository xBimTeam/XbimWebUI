Class: Prerequisities

Prerequisities

Constructor

new Prerequisities()

This is a structure reporting errors and warnings about prerequisities of xViewer. It is result of checkPrerequisities() static method.

Members

errors :Array.<string>

If this array contains any errors xViewer won't work at all or won't work as expected. You can use messages in this array to report problems to user. However, user won't probably be able to do to much with it except trying to use different browser. IE10- are not supported for example. The latest version of IE should be all right.

Type:
  • Array.<string>

noErrors :Array.<string>

If false xViewer won't work at all or won't work as expected. You can use messages in errors array to report problems to user. However, user won't probably be able to do to much with it except trying to use different browser. IE10- are not supported for example. The latest version of IE should be all right.

Type:
  • Array.<string>

noWarnings :Array.<string>

If false xViewer will work but it might be slow or may not support full functionality. Use warnings array to report problems.

Type:
  • Array.<string>

warnings :Array.<string>

If this array contains any warnings xViewer will work but it might be slow or may not support full functionality.

Type:
  • Array.<string>
